Crawljob

Used by seeds

Crawljob is used for define a job for harvesting. In addition to the parameters shown in the table below, the configuration also contains metadata.

Field Function
Deactivated Stop the job from running
Depth The depth to crawl
Max time How long the job should run
Max bytes Maximum amount of data to download
Schedule The schedule to use
Crawlconfig The crawlconfig to use

Deactivated

The switch is used to deativate the crawljob. With the switch set in the on position, the job will never run.

Depth

The depth decides how many levels of a web page to crawl. For example will a depth of the 3 mean that the crawler can visit 3 URL's way from the start page.

Max time

The maximum time in seconds that the job is allowed to run. The job will then run from the start time given in the schedule, until it reaches it's max time.

Max bytes

The maximum amount of data, given in bytes, that crawler can download. If exceeded the job should be stopped.

Schedule

Set the time for when the job should be run by defining a schedule.
The field is not required, but if the crawljob is missing a schedule the job will never run and must be started manually.

Crawlconfig

Choose what settings the crawljob should use be setting a crawlconfig. The crawlconfig includes settings for the web browser used for harvesting and politeness settings.
